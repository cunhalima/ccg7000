\langen{The \G7000{}. Games equipment adapts to a domestic colour TV set
	and consists in:
\begin{itemize}
	\item The \videopac{} computer, housed in a full alphanumeric keyboard
		  unit.
	\item Two joystick pads with an action button. The joysticks can be moved
		  in 8 distinguishable directions.
	\item A set of plug-in games cartridges, which are in fact \ROMs{}.
\end{itemize}}
\langbr{O \G7000{}. O equipamento funciona em televisores domésticos
	coloridos e consiste em:
	\begin{itemize}
		\item Computador \videopac{}, com um teclado alfanumérico.
		\item Dois controles com um botão de ação. As alavancas podem ser
		movidas em oito direções distintas.
		\item Um conjunto de cartuchos de jogos, que são, na verdade, \ROMs{}.
\end{itemize}}
